## Key concepts

Python is a high-level interpreted programming language.

It is beginner-friendly and widely used for:
- Automation
- Web development
- Data science
- Scripting

---

## Variables

Variables store data values.

Example:
name = "Huseyn"
age = 18

Python uses dynamic typing, meaning variable types are assigned automatically.

---

## Data Types

Main Python data types:
- str → text
- int → integers
- float → decimal numbers
- bool → True / False
- list → ordered mutable collection
- tuple → ordered immutable collection
- dict → key-value pairs
- set → unordered unique values

---

## Strings

Strings are text enclosed in quotes.

Useful methods:
- upper()
- lower()
- split()
- replace()

Formatting:
f"Hello, {name}"

---

## Numbers

Basic arithmetic operators:
- +
- -
- *
- /
- //
- %
- **

---

## Lists

Lists are mutable ordered collections.

Methods:
- append()
- remove()
- pop()
- sort()

---

## Tuples

Tuples are immutable ordered collections.
Used when data should not change.

---

## Dictionaries

Store key-value pairs.

Example:
student = {
    "name": "Huseyn",
    "age": 18
}

---

## Sets

Sets store unique values.
Useful for removing duplicates.

---

## Type Conversion

Convert between types using:
- int()
- float()
- str()
- list()

---

## Input and Output

Input:
input()

print()
